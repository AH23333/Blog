──[ 1.0 ]──────────────────────────────────────────────────────[ 测试概述 ]

本文用于验证图片渲染功能的正确性，涵盖不同尺寸、不同比例的图片类型。

所有图片均使用相对于 public/images/ 的标准化路径引用：

Blog\public\images\volume-0\image-render-test\

测试覆盖以下图片类型：

| 类型     | 尺寸      | 比例     | 用途       |
| -------- | --------- | -------- | ---------- |
| 小图标   | 100 x 100 | 1:1 方形 | 行内小图   |
| 流程图   | 400 x 300 | 4:3 标准 | 内容插图   |
| 横版横幅 | 800 x 200 | 4:1 超宽 | 页首横幅   |
| 竖版卡片 | 300 x 500 | 3:5 竖版 | 卡片式展示 |

──[ 2.0 ]──────────────────────────────────────────────[ 小尺寸图标 ]

小尺寸图标（100x100）适合用于行内标记或列表项前缀。

![小图标测试](Blog\public\images\volume-0\image-render-test\small-icon.svg)

小图标应居中显示，不拉伸变形，保持原始比例。

──[ 3.0 ]──────────────────────────────────────────────[ 中等尺寸流程图 ]

中等尺寸流程图（400x300）适合作为正文插图，辅助说明流程或架构。

![流程图测试](Blog\public\images\volume-0\image-render-test\medium-diagram.svg)

图片应自适应容器宽度，在移动端和桌面端都能正常显示。

──[ 4.0 ]──────────────────────────────────────────────[ 宽幅横版图 ]

宽幅横版图（800x200）适合用作页首横幅或分隔线装饰。

![横版横幅测试](Blog\public\images\volume-0\image-render-test\landscape-banner.svg)

超宽图片应能正确缩放，不会溢出容器边界。

──[ 5.0 ]──────────────────────────────────────────────[ 高幅竖版图 ]

竖版卡片图（300x500）适合用于人物介绍或侧边栏展示。

![竖版卡片测试](Blog\public\images\volume-0\image-render-test\portrait-card.svg)

竖版图片应在保持比例的前提下合理展示，不会被强制裁剪。

以下展示多张图片的混合布局，验证相邻图片的间距和排列。

![小图标测试](Blog\public\images\volume-0\image-render-test\small-icon.svg)

![流程图测试](Blog\public\images\volume-0\image-render-test\medium-diagram.svg)

![横版横幅测试](Blog\public\images\volume-0\image-render-test\landscape-banner.svg)

![竖版卡片测试](Blog\public\images\volume-0\image-render-test\portrait-card.svg)

多图连续排列时，图片之间应有适当间距，图片查看器应支持左右切换浏览。
